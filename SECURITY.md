# Security Policy

## Supported Versions

This project is actively maintained on the `main` branch.

## Reporting a Vulnerability

Please do not disclose security issues in public issues, pull requests, or
discussions.

Use one of the following private channels:

1. GitHub Security Advisory (preferred):
   `https://github.com/role-suite/role-serverpod/security/advisories/new`
2. If advisory creation is unavailable to you, open a normal issue with only
   non-sensitive metadata and request private maintainer contact.

When reporting, include:

- A clear description of the issue and impact
- Reproduction steps or proof of concept
- Affected versions/commits
- Suggested mitigation if known

We aim to acknowledge reports within 3 business days and provide a remediation
plan as quickly as possible.
